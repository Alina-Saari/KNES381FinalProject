Project Details / Background
Before this class I had no experience with Python so these projects took a lot of trial and error but I am much more comfortable with the program now.
I used python on three different platforms while creating these project, including the command bar with an ssh, Keggle and Jupyter notebook. The rocket was the first project that I completed where I learned the basics of python including ending lines, printing and basic code. Next was the Tic Tac Toe modifications where I learned and implemented boolean logic and conditional statements to improve the original code while using Jupyter notebook for the first time. Lastly was the data analysis on Keggle, where I learned how to use Keggle but also how to use python language to analysis data and create graphs.
